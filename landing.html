“Code The Web is a blog I made with one goal - to share my knowledge and inspire other people. Join me on a journey where I will guide you right through from getting your computer set up to making dynamic and interactive websites that look amazing.”

HTML

HTML is the language used to create websites. On Code The Web, I show you how to create your very own websites with HTML and give you a good understanding into the workings of HTML and the web.

CSS

CSS is the language used to style websites and make them look super-awesome and hipster. On Code The Web, I also teach you how to learn this cool language.

JavaScript

JavaScript it the language used to make websites do cool stuff. Without it, you wouldn't be able to do anything on the internet other than click on links. Learn how to make websites that do when I teach you JavaScript on Code The Web.
